# Assets do HUB (Fase 26)

Sprites opcionais para estruturas visuais dos POIs no HUB. Enquanto não houver arte, o jogo desenha **placeholders** por código (escada, balcão, retângulos por tipo).

Quando existirem, os arquivos esperados são:

- **dungeon_stairs.png** — escada descendo (entrada da dungeon)
- **shop_counter.png** — balcão da loja
- **inventory_shelf.png** — estante/baú (inventário), polish posterior
- **records_board.png** — painel de recordes, polish posterior

Estilo: pixel-art, tema apocalíptico do projeto.
